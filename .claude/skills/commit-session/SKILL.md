# Commit Session

## Instructions

今回のセッションで修正したファイルのみをcommit、pushする。

1. セッション中に修正したファイルのみを`git add`
2. 変更内容を簡潔に日本語で記述したコミットメッセージを作成
3. 現在のブランチにpush

## Notes

- コミットメッセージは日本語で簡潔に
- セッション中に修正したファイルのみを対象とする
- 全ての未コミット変更ではなく、今回の作業分のみ
